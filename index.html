Database NoSQL Ringan
di Atas SQLite

PocketDB adalah database NoSQL yang dibangun di atas SQLite untuk PHP. Tawarkan API mirip MongoDB, enkripsi data, dan indexing powerful tanpa server database terpisah.

download Mulai Sekarang code Lihat Repositori
PHP 
require_once 'src/Client.php';

$client = new Client('./data');
$users = $client->myDatabase->users;

// Insert data simpel
$users->insert([
    'name' => 'John Doe',
    'email' => 'john@example.com'
]);

// Query seperti MongoDB
$user = $users->findOne(['email' => 'john@example.com']);

Fitur Utama

PocketDB menggabungkan kesederhanaan file-based database dengan kekuatan query NoSQL.

Struktur & Komponen

PocketDB dirancang dengan struktur modular yang bersih, memisahkan tanggung jawab antara koneksi, koleksi, dan query.

1. Client

Titik masuk utama, manajemen path, dan koneksi pooling.

2. Database

Container koleksi, manajemen koneksi SQLite, dan enkripsi.

3. Collection

Operasi CRUD, ID generation, Event hooks, dan Indexing.

4. Cursor

Iterasi hasil query, pagination, sorting, dan population. 
// Database-level encryption
$db = new Database('./secure.db', [
    'encryption_key' => 'your-secret-key'
]);

// Event Hooks System
$users->on('beforeInsert', function($doc) {
    if (!filter_var($doc['email'], FILTER_VALIDATE_EMAIL)) {
        return false; // Cancel
    }
    $doc['created_at'] = date('Y-m-d H:i:s');
    return $doc;
});
Security & Flexibility

Keamanan & Hooks

Lindungi data sensitif Anda dengan enkripsi AES-256-CBC bawaan. Tidak hanya itu, sistem Hooks memungkinkan Anda menangkap event sebelum atau sesudah operasi database.

  • lock
    Encryption Support

    Enkripsi database maupun koleksi spesifik dengan searchable fields.

  • bolt
    Event-Driven

    Gunakan hook untuk validasi otomatis, logging, atau update relasi.

Use Cases

User Management

Validasi email otomatis menggunakan hooks sebelum insert. Mencegah perubahan email saat update profile.

E-Commerce

Manajemen stok yang aman. Transaksi otomatis mengurangi inventory saat order dibuat.

CMS / Blog

Workflow publishing dari draft ke published dengan timestamp otomatis.
